# 🧠 Interactive Python Quiz Game

A Python-based CLI quiz application with a 2-minute countdown timer, 10 multiple-choice questions, and instant feedback.

## Features
- 10 multiple-choice questions on Python & Computational Thinking
- 2-minute total timer using the `time` module
- Instant feedback after each answer
- Auto-termination when time runs out
- Final score display

## How to Run
```bash
python quiz_game.py
```

## Concepts Used
- `time` module for timer
- Functions & conditional statements
- Input handling & string methods
- Control flow with `exit()`
